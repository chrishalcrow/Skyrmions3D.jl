# Tutorial

This is quick guide to the main features of the Skyrmions3D package.

## Creating a Skyrmion

The main data type in the Skyrmions3D package is the [`Skyrmion`](@ref) type.
It consists of an underlying grid of a fixed resolution, along with the Skyrme or pion field on that grid.
